# Security Policy

## Supported versions

archview is pre-1.0; only the latest tagged release receives fixes.

## Reporting a vulnerability

Please report security issues privately to **dimas.eularix@gmail.com** rather
than opening a public issue. Include a description, affected version, and a
reproducer if possible. We aim to acknowledge reports within a few days.

Note: archview performs static analysis of your source and serves an
unauthenticated `/graph` UI by default — do not expose it on an untrusted
network without your own access control.
